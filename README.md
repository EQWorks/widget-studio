# widget-studio 

System to generate and control widgets, graduated from `ml-ui`. 

## Getting started
In `widget-studio` project root:
```shell
$ nvm use
$ yarn install
$ yarn start
```

Navigate to http://localhost:6008 to view the development storybook.
